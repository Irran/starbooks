Compared with the second edition, the change in the content of this edition is to change from IA32 and x86-64 to x86-64. The main updates are as follows: based on x86-64, a large number of rewrite codes, the first introduction to machine-level support for programs that process floating-point data. The processor architecture was modified to support 64-bit word and operation design. More functional units and more complex control logic are introduced to make program performance model prediction based on program data flow representation more reliable.
